# TaskWave AI 🌊

> **Ускорьте работу над проектом: мгновенное превращение аудио-инсайтов в управляемые задачи.**

TaskWave анализирует ваши аудиовстречи и автоматически формирует архитектуру задач, эпики и критерии приёмки на нескольких языках.

---

## Как работает

```
Запись митинга (RU / EN / JP)
      │
      ▼
  AI Распознавание     — локальная расшифровка речи (Whisper)
      │
      ▼
  AI Корректор         — исправление технического жаргона
      │
      ▼
  LLM Архитектор       — генерация эпиков, задач, DoD, AC
      │
      ▼
 Готовый бэклог        — экспорт в Jira / Linear / Trello
```

---

## Текущий статус

### ✅ Фаза 1 — AudioPM Local (готово)
CLI-движок для полностью офлайн работы.
- Локальная расшифровка (Whisper.net)
- Интерактивный TUI интерфейс

### ✅ Фаза 2 — TaskWave API (готово)
ASP.NET Core Minimal API для интеграции.
- SSE стриминг прогресса в реальном времени
- Управление жизненным циклом заданий

### ✅ Фаза 3 — TaskWave Web (готово)
SvelteKit фронтенд премиум-класса.
- Drag-and-drop загрузка
- Визуализация архитектуры задач (Hierarchy / List / Epics)

### ✅ Фаза 4 — Localization & UX (готово)
Полная поддержка международного использования.
- **RU / EN / JP**: Интерфейс и генерация контента полностью локализованы
- Улучшенная UX-центричность и премиальный дизайн

## Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/Piryutko/TaskWave-AI.git
cd TaskWave-AI

# 2. Запустить API (порт 5000)
cd TaskWave.Api
dotnet run --urls "http://localhost:5000"

# 3. Запустить фронтенд (порт 5173)
cd ../taskwave-web
npm install
npm run dev

# 4. Открыть браузер
# http://localhost:5173
```

> **Требования:** .NET 8 SDK, Node.js 18+, [Ollama](https://ollama.ai/) (`ollama pull llama3.2`)

## Технологии

| Слой | Технология |
|---|---|
| CLI движок | .NET 8, C# 12 |
| AI Распознавание | Whisper.net (локально) |
| LLM | Ollama HTTP API |
| HTTP API | ASP.NET Core Minimal API |
| Стриминг | Server-Sent Events (SSE) |
| Фронтенд | SvelteKit + TypeScript |
| Стили | Tailwind CSS v4 (Glassmorphism) |
